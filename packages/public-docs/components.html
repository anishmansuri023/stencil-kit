MyButton

A customizable button component with multiple variants and sizes.

Live Demo

Variants

Primary Secondary Danger

Sizes

Small Medium Large

States

Enabled Disabled

Props

Prop Type Default Description
variant 'primary' | 'secondary' | 'danger' 'primary' Button style variant
size 'small' | 'medium' | 'large' 'medium' Button size
disabled boolean false Disabled state

Usage Examples

HTML 
<my-button variant="primary" size="medium">Click Me</my-button>
<my-button variant="secondary" size="small">Secondary</my-button>
<my-button variant="danger" disabled>Disabled</my-button>
React 
import { MyButton } from 'react-library';

function App() {
  return (
    <>
      <MyButton variant="primary" size="medium">Click Me</MyButton>
      <MyButton variant="secondary" size="small">Secondary</MyButton>
      <MyButton variant="danger" disabled>Disabled</MyButton>
    </>
  );
}
Vue 
<template>
  <my-button variant="primary" size="medium">Click Me</my-button>
  <my-button variant="secondary" size="small">Secondary</my-button>
  <my-button variant="danger" disabled>Disabled</my-button>
</template>
Angular 
<my-button variant="primary" size="medium">Click Me</my-button>
<my-button variant="secondary" size="small">Secondary</my-button>
<my-button variant="danger" [disabled]="true">Disabled</my-button>

MyCard

A card container component with header, content, and footer sections.

Live Demo

Card with low elevation

Action

Card with medium elevation

Action

Card with high elevation

Action

Props

Prop Type Default Description
cardTitle string undefined Card title text
subtitle string undefined Card subtitle text
elevation 'low' | 'medium' | 'high' 'medium' Shadow depth/elevation

Slots

Slot Description
(default) Main card content
footer Card footer content

Usage Examples

HTML 
<my-card card-title="Card Title" subtitle="Subtitle" elevation="medium">
  <p>Card content goes here</p>
  <div slot="footer">
    <my-button variant="primary">Action</my-button>
  </div>
</my-card>
React 
import { MyCard, MyButton } from 'react-library';

<MyCard cardTitle="Card Title" subtitle="Subtitle" elevation="high">
  <p>Card content goes here</p>
  <div slot="footer">
    <MyButton variant="primary">Action</MyButton>
  </div>
</MyCard>
Vue 
<template>
  <my-card card-title="Card Title" subtitle="Subtitle" elevation="medium">
    <p>Card content goes here</p>
    <div slot="footer">
      <my-button variant="primary">Action</my-button>
    </div>
  </my-card>
</template>
Angular 
<my-card cardTitle="Card Title" subtitle="Subtitle" elevation="medium">
  <p>Card content goes here</p>
  <div slot="footer">
    <my-button variant="primary">Action</my-button>
  </div>
</my-card>
