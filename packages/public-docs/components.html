MyInput

A text input field with label, placeholder, error state, and multiple input types.

Live Demo

Props

PropTypeDefaultDescription
labelstring''Label above the input
placeholderstring''Placeholder text
valuestring''Input value
type'text' | 'email' | 'password' | 'number' | 'tel''text'HTML input type
errorstring''Error message shown below
disabledbooleanfalseDisabled state
requiredbooleanfalseShows * on label

Events

EventDetailDescription
myChangestringFires on every keystroke with the current value
myBlurvoidFires when input loses focus

Usage Examples

HTML
<my-input label="Email" type="email" placeholder="you@example.com" required></my-input>
<my-input label="Password" type="password" error="Too short"></my-input>
React
import { MyInput } from '@stencil-kit/react';

function App() {
  const [value, setValue] = useState('');
  return (
    <MyInput
      label="Email"
      type="email"
      value={value}
      onMyChange={e => setValue(e.detail)}
    />
  );
}
Vue
<template>
  <my-input label="Email" type="email" :value="email" @my-change="email = $event.detail" />
</template>
Angular
<my-input label="Email" type="email" [value]="email" (myChange)="email = $event.detail"></my-input>

A dialog overlay with header, scrollable body, footer slot, and backdrop dismiss.

Props

PropTypeDefaultDescription
openbooleanfalseControls visibility
modal-titlestring''Title in the header
size'small' | 'medium' | 'large''medium'Modal width

Slots

SlotDescription
(default)Modal body content
footerFooter buttons / actions

Events

EventDescription
myCloseFires when modal is dismissed

Usage Examples

HTML
<my-button onclick="document.querySelector('my-modal').open = true">Open</my-button>

<my-modal modal-title="Confirm Action" size="small">
  <p>Are you sure you want to proceed?</p>
  <div slot="footer">
    <my-button variant="secondary">Cancel</my-button>
    <my-button variant="primary">Confirm</my-button>
  </div>
</my-modal>
React
import { MyModal, MyButton } from '@stencil-kit/react';

function App() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <MyButton onClick={() => setOpen(true)}>Open Modal</MyButton>
      <MyModal modalTitle="Confirm" open={open} onMyClose={() => setOpen(false)}>
        <p>Modal content here</p>
        <div slot="footer">
          <MyButton variant="primary" onClick={() => setOpen(false)}>OK</MyButton>
        </div>
      </MyModal>
    </>
  );
}
Vue
<template>
  <my-button @click="open = true">Open Modal</my-button>
  <my-modal modal-title="Confirm" :open="open" @my-close="open = false">
    <p>Modal content here</p>
    <div slot="footer">
      <my-button variant="primary" @click="open = false">OK</my-button>
    </div>
  </my-modal>
</template>
Angular
<my-button (click)="open = true">Open Modal</my-button>
<my-modal modalTitle="Confirm" [open]="open" (myClose)="open = false">
  <p>Modal content here</p>
  <div slot="footer">
    <my-button variant="primary" (click)="open = false">OK</my-button>
  </div>
</my-modal>

MyDropdown

A custom select dropdown with keyboard-friendly design, label, error state, and disabled support.

Live Demo

Props

PropTypeDefaultDescription
optionsArray<{label, value}> | string[]Dropdown options (JSON string in HTML)
valuestring''Selected value
placeholderstring'Select an option'Placeholder text
labelstring''Label above the dropdown
disabledbooleanfalseDisabled state
errorstring''Error message

Events

EventDetailDescription
myChangestringSelected option value

Usage Examples

HTML
<my-dropdown
  label="Framework"
  options='[{"label":"React","value":"react"},{"label":"Vue","value":"vue"}]'
></my-dropdown>
React
import { MyDropdown } from '@stencil-kit/react';

const options = [
  { label: 'React', value: 'react' },
  { label: 'Vue', value: 'vue' },
];

function App() {
  const [val, setVal] = useState('');
  return (
    <MyDropdown
      label="Framework"
      options={options}
      value={val}
      onMyChange={e => setVal(e.detail)}
    />
  );
}
Vue
<template>
  <my-dropdown label="Framework" :options="options" :value="val" @my-change="val = $event.detail" />
</template>
Angular
<my-dropdown label="Framework" [options]="options" [value]="val" (myChange)="val = $event.detail"></my-dropdown>
