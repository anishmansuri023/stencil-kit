Vue 3 Installation

Add Stencil components to your Vue 3 application

Installation

Terminal 
npm install @your-org/vue-library

Setup

Register the component library plugin in your main.ts file:

main.ts 
import { createApp } from 'vue';
import App from './App.vue';
import { ComponentLibrary } from '@your-org/vue-library';

const app = createApp(App);

// Register Stencil components globally
app.use(ComponentLibrary);

app.mount('#app');

Vite Configuration

If using Vite, configure it to recognize custom elements:

vite.config.ts 
import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';

export default defineConfig({
  plugins: [
    vue({
      template: {
        compilerOptions: {
          // Treat all tags starting with 'my-' as custom elements
          isCustomElement: (tag) => tag.startsWith('my-')
        }
      }
    })
  ]
});

Event Handling

Events 
<template>

    <p>Count: {{ count }}</p>
    <my-button 
      variant="primary" 
      @click="count++"
    >
      Increment
    </my-button>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue';

const count = ref(0);
</script>

Reactive Props

Reactive Example 
<template>
  <my-button 
    :variant="buttonVariant" 
    :disabled="isDisabled"
    @click="handleClick"
  >
    {{ buttonText }}
  </my-button>
</template>

<script setup lang="ts">
import { ref } from 'vue';

const buttonVariant = ref<'primary' | 'secondary' | 'danger'>('primary');
const isDisabled = ref(false);
const buttonText = ref('Click Me');

const handleClick = () => {
  console.log('Button clicked!');
};
</script>

Complete Example

Full Example 
<template>
  <div style="padding: 20px">
    <h1>Dashboard</h1>
    
    <my-card 
      card-title="Notifications" 
      :subtitle="`You have ${notifications} new notifications`"
      elevation="medium"
    >
      <p>Check your latest updates</p>
      
      <div slot="footer">
        <my-button 
          variant="primary" 
          size="small"
          @click="notifications = 0"
        >
          Mark All Read
        </my-button>
        <my-badge color="warning">{{ notifications }}</my-badge>
      </div>
    </my-card>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue';

const notifications = ref(3);
</script>
