Why Choose Stencil Design System?

🚀

Framework Agnostic

Works seamlessly with React, Vue, Angular, or vanilla JavaScript. No lock-in.

📦

Tiny Bundle Size

Only ~15KB gzipped for the entire library. Lazy-loaded components.

♿

Accessible

WCAG compliant components with keyboard navigation and screen reader support.

🎨

Customizable

CSS custom properties for easy theming. Style to match your brand.

⚡

High Performance

Built with StencilJS for optimal runtime performance and small footprint.

📘

TypeScript First

Full TypeScript support with auto-generated type definitions.

Installation

Choose your framework and follow the installation guide:

⚛️

React

npm install @your-org/react-library
💚

Vue 3

npm install @your-org/vue-library
🅰️

Angular

npm install @your-org/angular-library
📄

Vanilla JS

npm install @your-org/stencil-library

Quick Example

Here's how easy it is to use our components:

React 
import { MyButton, MyCard } from '@your-org/react-library';

function App() {
  return (
    <MyCard cardTitle="Welcome" elevation="medium">
      <p>Get started with our component library</p>
      <div slot="footer">
        <MyButton variant="primary">Get Started</MyButton>
      </div>
    </MyCard>
  );
}
Vue 3 
<template>
  <my-card card-title="Welcome" elevation="medium">
    <p>Get started with our component library</p>
    <div slot="footer">
      <my-button variant="primary">Get Started</my-button>
    </div>
  </my-card>
</template>
Angular 
<my-card cardTitle="Welcome" elevation="medium">
  <p>Get started with our component library</p>
  <div slot="footer">
    <my-button variant="primary">Get Started</my-button>
  </div>
</my-card>
Vanilla JS 
<script type="module" src="https://unpkg.com/@your-org/stencil-library"></script>

<my-card card-title="Welcome" elevation="medium">
  <p>Get started with our component library</p>
  <div slot="footer">
    <my-button variant="primary">Get Started</my-button>
  </div>
</my-card>
