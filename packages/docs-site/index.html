Quick Start

Terminal 
# Clone the repository
git clone <your-repo-url>
cd my-monorepo

# Install dependencies
npm install

# Build all libraries
npm run build

# Run all demo apps
npm run dev

The npm run dev command will automatically open 4 browser tabs with all demo applications.
