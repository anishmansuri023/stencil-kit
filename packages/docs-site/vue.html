Vue 3 Integration

Vue plugin with global component registration

Installation

The Vue library is included at packages/vue-library.

Terminal
npm run build

Register Plugin

main.ts
import { createApp } from 'vue';
import App from './App.vue';
import { ComponentLibrary } from 'vue-library';

const app = createApp(App);
app.use(ComponentLibrary);
app.mount('#app');

Usage

App.vue
<template>
  <my-button variant="primary" size="large">Click Me</my-button>
  <my-card card-title="Welcome" subtitle="Get started" elevation="high">
    <p>Card content</p>
    <div slot="footer">
      <my-badge color="success">New</my-badge>
    </div>
  </my-card>
</template>
