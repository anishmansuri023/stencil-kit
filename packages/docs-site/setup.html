Setup Guide

Complete installation and setup instructions

Prerequisites

  • Node.js: v18 or higher
  • npm: v9 or higher
  • Git: Latest version

Installation

1. Clone the Repository

Terminal 
git clone <your-repo-url>
cd my-monorepo

2. Install Dependencies

Terminal 
npm install

This will install all dependencies for all packages in the monorepo using npm workspaces.

3. Build the Libraries

Terminal 
npm run build

This builds all libraries in the correct dependency order using Lerna and Nx.

Development

Run All Demo Apps

Terminal 
npm run dev

This will:

  1. Start all 4 dev servers in parallel
  2. Automatically open 4 browser tabs:
    • HTML: http://localhost:3000/html-app/
    • Angular: http://localhost:4200/
    • React: http://localhost:5173/
    • Vue: http://localhost:5174/

Run Individual Apps

Terminal 
# React app only
cd packages/react-app
npm run dev

# Vue app only
cd packages/vue-app
npm run dev

# Angular app only
cd packages/angular-workspace
npm run dev

# HTML app only
cd packages/html-app
npm run dev

Available Commands

Command Description
npm run build Build all libraries (stencil, react, vue, angular)
npm run build:libs Build only core libraries (not apps)
npm run build:all Build everything including apps
npm run dev Run all dev servers and open browsers
npm run test Run all tests
npm run clean Clean all node_modules
npm run graph View dependency graph

Troubleshooting

Build Errors

If you encounter build errors:

Terminal 
# Clean all node_modules
npm run clean

# Reinstall dependencies
npm install

# Rebuild everything
npm run build

Port Conflicts

If ports are already in use, you can change them in the respective package.json files or kill the processes:

Terminal 
# Find process using port
lsof -i :4200

# Kill process
kill -9 <PID>

TypeScript Errors

Make sure all libraries are built before running apps:

Terminal 
npm run build
